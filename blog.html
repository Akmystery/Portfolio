How to secure our facebook account

ကြ်န္ေတာ္တို႔ဟာ facebook ကိုရည္ရြယ္ခ်က္အမ်ိဳးမ်ိဳး နဲ႔ ေန႔စဥ္ သံုးပါတယ္။ ဒါေပမယ္႔ မိမိတို႔ facebook account ေတြရဲ႔ security ပိုင္းကို ဂရုစိုက္ဖို႔ ေမ႔ေနတတ္ၾကတယ္။ ဒီိလို ေမ႔ေနတတ္ၾကတဲ႔ အတြက္ အခုတစ္ေလာ ေခတ္စားေနတဲ႔ facebook account hacker ေတြနဲ႔ ေတြ႔ရင္ မိမိရဲ႕ account ေလးေတြပါသြားနိုင္ပါတယ္။ အဲ႔ဒိေတာ႔ ဒီ post ေလးမွာ ကြ်န္ေတာ္ facebook account ကို လံုလံုျခံုျခံု သံုးနိုင္ဖို႔ အေျခခံနည္းလမ္းေလးေတြ ၿပီးေတာ့ facebook ကေနျပီးေတာ႔ လုပ္ေပးထားတဲ႔ extra account security နည္းလမ္း (၃) ခုအေၾကာင္း ေျပာျပေပးသြားမွာ ျဖစ္ပါတယ္။ အခု post ေလးကို ဖတ္ျပီးတာနဲ႔ သူငယ္ခ်င္းအေပါင္း တို႔လည္း ခ်က္ခ်င္း မိမိ account ကို secure ျဖစ္ေအာင္ လုပ္မယ္လို႔ ေမွ်ာ္လင္႔ပါတယ္။ လုပ္လဲ လုပ္ေစခ်င္ပါတယ္။

ပထမဆံုးအေနနဲ႔ facebook account ကို လံုျခံဳစြာသံုးႏိုင္ဖို႔ အေျခခံေဆာင္ရန္၊ ေရွာင္ရန္ အခ်က္ေတြကို ေျပာျပေပးသြားမွာ ျဖစ္ပါတယ္။

  1. Password ကို strong ျဖစ္တာသံုးပါ။ မိမိရဲ႕ နာမည္ေတြ၊ ေမြးေန႔ေတြ၊ 12345678 တို႔၊ iloveu တို႔၊ asdfghjkl တို႔ို လြယ္တဲ႔ password ေလးေတြ လံုး၀ မသံုးပါနဲ႔။ Facebook account ရဲ႕ password နဲ႔ gmail account ရဲ႕ password ကို လံုး၀ မတူပါေစနဲ႔။ ကြဲျပားျခားနားတာ သံုးပါ။
  2. Facebook မွာရွိတဲ႔ Link ေတြကို ေသေသခ်ာခ်ာ မသိရင္ မနွိပ္ပါနဲ႔။ ကိုယ္ရင္းနွီးတဲ႔ သူဆီက link ေတြ ေရာက္လာရင္ေတာင္ ပို႔တဲ႔ သူကို ေသခ်ာေမးၾကည္႔ျပီးွ နွိပ္သင္႔မွနွိပ္ပါ။
  3. မိမိနဲ႔ မသိတဲ႔ သူ ေတြ လာ add ရင္ ျဖစ္နိုင္ရင္ လက္မခံပါနဲ႔။
  4. အမ်ားသံုး ကြန္ပ်ဴတာ ေတြ မွာ facebook account ကို log in မ၀င္ပါနဲ႔။ ျပီးေတာ႔ public wifi သံုးျပီး facebook account ကို log in မ၀င္ပါနဲ႔။ ဖုန္းထဲမွာ log in ၀င္ျပီးသား account ကို public wifi နဲ႔ သံုးတာကေတာ႔ ျပႆနာ မရိွပါဘူး။
  5. www.facebook.com ကမဟုတ္ဘဲ နဲ႔ တစ္ျခား website နဲ႔ မိမိရဲ႕ username နဲ႔ password ကို ေတာင္းရင္ website အေၾကာင္းကို ေသေသခ်ာခ်ာ ေလ႔လာျပီး သိမွ ေပးပါ။ ျဖစ္နိုင္ရင္မေပး ပါနဲ႔။
  6. မိမိ account မွာတင္ထားတဲ့ email address နဲ႔ phone number တို႔ကို only me လုပ္ထားပါ။

အထက္ပါနည္းလမ္းေတြက အေျခခံ ေဆာင္ရန္ ေရွာင္ရန္ေလးေတြပါ။ အခု ကြ်န္ေတာ္ facebook ကလုပ္ေပးထားတဲ႔ security နည္းလမ္း ၃ ခု အေၾကာင္း ေျပာျပမွာ ျဖစ္ပါတယ္။

  1. Get alerts about unrecognized log-ins.
    ဒီ feature ေလးက မိမိ facebok account ကို ကိုယ္သံုးေနက် phone တို႔ computer တို႔က မဟုတ္ဘဲ တစ္ျခား device တစ္ခုခုက ၀င္ဖို႔ၾကိဳးစားတိုင္း မိမိကို အသိေပးမွာျဖစ္ပါတယ္။ ထိုအသိေပးခ်က္ရ တာနဲ႔ မိမိရဲ႕ account password ကို ခ်က္ခ်င္း reset လုပ္လို႔ရ ေအာင္ လဲ လုပ္ထားေပးပါတယ္။
  2. Use two factor authentication.
    ဒီ feature ေလးကေတာ႔ မိမိရဲ႕ account ထဲကို log-in ၀င္ဖို႔ မိမိရဲ႕ account ရဲ႕ password အျပင္ မိမိ set up လုပ္ထားတဲ႔ device ကို ပို႔တဲ႔ security code ကိုပါ လိုအပ္မွာျဖစ္ပါတယ္။ ကြ်န္ေတာ္ကေတာ႔ ဖုန္းနံပါတ္ ကို အသံုးျပဳျပီး set up လုပ္လုိ႔ရတဲ႔ နည္းကို ေျပာျပသြားမွာ ျဖစ္ပါတယ္။ တစ္ျခားနည္းလမ္းေတြလဲ စမ္းၾကည့္ႏိုင္ပါတယ္ ။ ဒီ feature ေလးကို set up လုပ္ထားပါက သင္႔ရဲ႕ password ကို တစ္ျခားသူ ရသြားရင္ေတာင္ သင္႔ရဲ႕ ဖုန္းမရွိ ဘဲနဲ႔ facebook ထဲကို log-in ၀င္လို႔မရပါဘူး။ တစ္ခုေတာ့ ရွိပါတယ္။ မိမိကိုယ္တိုင္လဲ မိမိသံုးေနက် device မဟုတ္ပဲ တစ္ျခား computer ေတြ phone ေတြကေန မိမိ account ထဲကို log in ဝင္ခ်င္ရင္လဲ မိမိရဲ႕ ဖုန္းက အနားမွာ ရွိေနဖို႔ လိုပါတယ္။
  3. Choose 3 to 5 friends to contact if you are locked out.
    ဒီ feature ေလးကေတာ႔ ကံဆိုးျပီး သင္႔ account ထဲကို log-in ၀င္လို႔မရေတာ႔ရင္ သင္ေရြးခ်ယ္ထားတဲ႔ သူငယ္ခ်င္းေတြ ဆီက အကူအညီနဲ႔ သင္႔ရဲ႕ account ကို recover (ျပန္ယူ) နိုင္ပါတယ္။ အဲ႔ဒီလို သူငယ္ခ်င္း ေရြးခ်ယ္ရာ မွာ သင္နဲ႔ အျပင္မွာ တကယ္ ရင္းနွီးတဲ႔ သူငယ္ခ်င္း ၿပီးေတာ့ account တစ္ကိုခုထဲကိုပဲ စြဲစြဲၿမဲၿမဲ သံုးတဲ့ သူငယ္ခ်င္းကို ေရြးခ်ယ္ေစခ်င္ပါတယ္။

အားလံုး facebook ကို လံုလံု ျခံုျခံု သံုးနိုင္ၾကပါေစ။
